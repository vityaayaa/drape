# Drape

## Что это
Веб-приложение (PWA) для визуализации пиксель-арт мозаики на стенах
помещения. Пользователь задаёт параметры комнаты, добавляет объекты,
загружает изображение, которое пикселизируется и раскладывается по
стенам в 3D. На выходе — схема с цветами и количеством плиток
для реального воплощения.

## Пользователь
Один человек. Без аккаунтов, логинов и облачных сохранений.
Сохранение проекта — экспорт JSON-файла и его повторная загрузка.

## Ключевой сценарий
1. Задать параметры комнаты (ширина и высота стен)
2. Опционально добавить маски-препятствия (лестница, дверь,
   окно и т.п.) — прямоугольные области, исключающие плитки из расчёта
3. Загрузить фото, назначить на стены:
   панорама на все стены или для каждой стены своё (или ничего)
4. Настроить пикселизацию: размер пикселя, плотность, расположение
5. Базовая цветокоррекция: яркость, контраст, насыщенность, температура, тени/света
6. Посмотреть в 3D, покрутить под разными углами
7. Режим укладки — пошаговая инструкция для кладчика
8. Скачать схему + спецификацию цветов (сколько плиток какого цвета)

## В скоупе
- PWA (без APK и нативных приложений)
- Экспорт/импорт файла проекта (JSON)
- Оптимизация обработки изображений (Web Worker + OffscreenCanvas)
- AR — под вопросом, решается в Сессии 4

## Вне скоупа
- Нативные приложения (APK и т.п.)
- Онлайн-заказ плиток
- Аккаунты и облачные сохранения
- Серверная обработка (всё в браузере)

## Архитектурное направление
Всё на клиенте, без бэкенда и БД. Обработка изображений —
Web Worker + OffscreenCanvas, чтобы не нагружать главный поток.
Конкретный стек выбирается в Сессии 6.

## Критерии готовности
Можно пройти весь сценарий от начала до конца.

## Что зафиксировано до старта
- Форма комнаты задаётся вручную: пользователь вводит количество
  стен и их размеры
- Размер плитки произвольный, задаётся пользователем
- Плитки могут быть объёмными (например, кирпич) — это влияет
  на полезную площадь стены и геометрию углов
- Не обязательно покрывать все стены — может быть только
  часть стен (например, три из четырёх)
