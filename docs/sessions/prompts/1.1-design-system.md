## Промпт

Прочитай файл `docs/sessions/state.md` — там контекст проекта и полный список проблем.

Ты дизайнер мобильного PWA-приложения **Drape** (укладка плитки). Наша задача сегодня — выработать **единую дизайн-систему**, которая будет применена ко всем вкладкам в последующих сессиях.

**Что сейчас есть:**
- Тёмная glassmorphism тема (фиолетово-синие градиенты, стеклянные карточки)
- Разные стили у Комнаты и Фото — нет единства
- Нет зафиксированной палитры, типографики, отступов

**Задача:**
Предложи и обсуди со мной дизайн-систему для всего приложения:
1. **Направление**: остаёмся в glassmorphism или меняем? Покажи 2-3 альтернативы с описанием ощущения.
2. **Цветовая палитра**: основной фон, поверхности карточек, акцент, текст (4 уровня), ошибка, успех
3. **Типографика**: размеры и веса для заголовков, body, label, hint, caption
4. **Радиусы и тени**: единые значения для карточек, кнопок, инпутов
5. **Отступы**: система (4px base grid или 8px?)
6. **Кнопки**: primary, secondary, ghost, danger — внешний вид каждого
7. **Инпуты**: стиль полей ввода (сейчас разные в разных вкладках)
8. **Иконки в навбаре**: предложи иконки для Комнаты, Фото, 3D, Схемы, Укладки
