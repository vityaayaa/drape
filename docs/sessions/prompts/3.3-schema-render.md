## Промпт

Прочитай `docs/sessions/state.md` — там контекст проекта и **дизайн-решения сессий 3.1 и 3.2** (split-screen, развёртка, легенда, экспорт).

Мы реализуем вкладку **«Схема»**: верхнюю часть split-screen — интерактивную развёртку стен с цветными плитками.

**Прочитай перед началом:**
- `src/components/export/ExportTab.jsx` — текущая заглушка
- `src/store/projectStore.js` — структура данных
- `src/utils/pixelizerGeometry.js` — расчёт сетки плиток (`tileRect`, `isFullyInsideMask`)
- `src/utils/pixelizerRenderer.js` — рендер canvas (для понимания паттернов)
- `src/utils/computeWallPositions.js`

**Структура данных:**
- `walls[]` — стены с размерами и масками
- `tile` — глобальные параметры плитки (ширина, высота, шов, grout_color)
- `pixelizer.tileColors` — `wallId → { 'col_row': '#rrggbb' }` — цвет каждой плитки после пикселизации

**Задачи:**

**З1 — Компонент `SchemaRenderer`**
Новый файл `src/utils/schemaRenderer.js`. Принимает `{ walls, tile, tileColors }`, рисует развёртку на `<canvas>`.

- Расположение стен: горизонтально одной строкой, пропорционально реальной ширине (высота стен нормализована к единому значению для вмещения в экран)
- Каждая плитка: залитый прямоугольник цветом из `tileColors`, шов — цвет `grout_color`
- Маски-препятствия: прямоугольник поверх плиток (стиль из сессии 3.1)
- Название стены: подпись под каждой стеной
- Нумерация плиток: номер цвета (индекс в палитре) внутри клетки — только если плитка достаточно большая (минимум 12px)

**З2 — Палитра цветов**
Функция `buildPalette(tileColors): Array<{ index, hex, count, byWall }>`.
- Собрать уникальные цвета по всем стенам
- Подсчитать количество плиток каждого цвета (общее и по стенам)
- Отсортировать по убыванию count
- Индекс = порядковый номер в этой сортировке (1, 2, 3...)

**З3 — Компонент `SchemaView`**
`src/components/export/SchemaView.jsx` — верхняя половина split-screen.
- `<canvas>` с рендером развёртки через `SchemaRenderer`
- Зум и пан через `react-zoom-pan-pinch` (уже в проекте)
- Кнопка сброса зума (иконка `Maximize2`, Lucide) — в стиле дизайн-системы
- Пустое состояние: если `tileColors` пусто — `EmptyState` с иконкой `PenLine` и кнопкой «→ Перейти в Фото»

**З4 — Интеграция в `ExportTab`**
Заменить заглушку на полноценный split-screen:
- Верхняя часть: `<SchemaView>` (высота регулируется через state)
- Разделитель: `<div>` с drag-handler (touch + mouse)
- Нижняя часть: пока пустой `<div>` с placeholder текстом «Легенда — следующий этап»
- Пропорции по умолчанию из сессии 3.1

**Важно:**
- `react-zoom-pan-pinch` уже в проекте — использовать, не добавлять зависимости
- Рендер тяжёлый при большом числе плиток — использовать `useMemo` для palette, пересчитывать canvas только при изменении `tileColors` или `walls`
