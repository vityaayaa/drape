## Сессия 3.1 — Дизайн: layout и развёртка

Используй скилл `/ui-ux-pro-max` для принятия дизайн-решений.

Реши самостоятельно (опираясь на Dark Glow из state.md и EXPORT.md):

**Split-screen layout:**
- Пропорции по умолчанию: 60% развёртка / 40% легенда
- Разделитель (handle): горизонтальная полоска 4px, `background: rgba(255,255,255,0.08)`, с иконкой `⋯` в центре, touch target 24px
- Минимум верхней части: 30% экрана, нижней: 20% экрана

**Развёртка стен:**
- Масштаб: все стены одинаковой высоты, ширина пропорциональна реальной ширине стены
- Зазор между стенами: 16px, вертикальная линия-разделитель `rgba(255,255,255,0.07)`
- Название стены: подпись снизу под стеной, 10px, `#64748b`
- Маски-препятствия: затемнённый прямоугольник `rgba(0,0,0,0.45)` + диагональная штриховка `rgba(255,255,255,0.06)` (SVG pattern)

**Плитки:**
- Номер цвета: числовой индекс (1, 2, 3...), по центру клетки, белый текст, размер шрифта = min(tileSize * 0.4, 11px)
- При клетке < 8px — номер не рисовать
- Цвет шва: берём из настроек (`grout_color`)

**Жесты:**
- `react-zoom-pan-pinch` (уже в проекте) на развёртке, независимо от нижней части
- Кнопка reset zoom: иконка `Maximize2` (Lucide, 18px), 36×36, `background: rgba(14,16,24,0.9)`, `border: 1px solid rgba(255,255,255,0.1)`, позиция: правый нижний угол развёртки, отступ 8px

**Пустое состояние развёртки:**
- Компонент `EmptyState` (уже в проекте: `src/components/shared/EmptyState.jsx`) с иконкой `PenLine`, текстом «Пикселизация не запущена», subtitle «Перейдите во вкладку Фото и нажмите «Пикселизировать»», кнопка «→ Перейти в Фото»

**Toolbar вкладки:**
- Нет отдельного toolbar — кнопка «Скачать SVG» живёт в нижней части (легенда), прилипает к низу экрана

Запиши результат в `docs/sessions/state.md`:

## Сессия 3.2 — Дизайн: легенда и экспорт

Используй скилл `/ui-ux-pro-max`.

Реши самостоятельно:

**Шапка нижней части (фиксированная):**
- Одна строка: «[W]×[H]мм · шов [G]мм» + квадратик цвета шва + Запас: `[input]%`
- Поле запаса: `type="text" inputMode="numeric"`, ширина 48px, высота 32px, значение по умолчанию 10

**Список цветов:**
- Строка: квадратик 14×14 (borderRadius 3) + порядковый номер `#N` + hex + «N шт.» + при запасе > 0: `→ M шт.` (`#64748b`)
- Разбивка по стенам: раскрывается по тапу на строку (chevron справа), отступ 24px слева
- Сортировка: по убыванию количества
- Шрифт строки: 13px, `#f1f5f9`

**Кнопка «Скачать SVG»:**
- Primary кнопка, полная ширина, высота 48px, прилипает к низу (position sticky bottom), выше safe area
- Иконка `Download` (Lucide, 16px) слева от текста

**Диалог настроек экспорта:**
- Bottom sheet (стиль как `WallsSheet` в Фото)
- Две radio-кнопки: «На экран» / «Реальный размер 1:1»
- Toggle «Рисовать швы» (disabled при grout_width = 0)
- Primary кнопка «Скачать» (со спиннером при генерации)

**Состояние загрузки:**
- Спиннер через CSS `::after` на кнопке (как у «Пикселизировать» в Фото)
- Кнопка disabled во время генерации

**Создай:**

`src/utils/schemaSVGBuilder.js` — функция `buildSchemaSVG({ walls, tile, tileColors, palette, options })`:
- `options.scale: 'fit' | 'real'` — «На экран» или «Реальный размер 1:1»
- `options.drawGrout: boolean`
- `options.surplusPercent: number`
- При `scale: 'real'`: `width`/`height` в мм (`width="1200mm"`)
- При `scale: 'fit'`: `viewBox`, `width="100%"`
- При > 10 000 плиток: `<symbol>`/`<use>` для оптимизации (каждый уникальный цвет = один `<symbol>`)
- Содержимое SVG: развёртка стен + легенда под ней

`src/components/export/ExportDialog.jsx` — bottom sheet:
- Заголовок «Настройки экспорта»

Функция `downloadSVG(svgString, filename)`:
- `Blob` → временная ссылка → `a.click()` → `URL.revokeObjectURL`
