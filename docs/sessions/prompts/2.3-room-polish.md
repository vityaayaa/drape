## Промпт

Прочитай `docs/sessions/state.md` — там дизайн-решения сессии 1.1 (дизайн-система) и 1.2 (Комната UI), а также результаты предыдущих кодинг-сессий.

Работаем с вкладкой **«Комната»** и общим App.jsx.

**Прочитай перед началом:**
- `src/components/room/RoomTab.jsx`
- `src/components/room/WallCard.jsx`

**Задачи:**

**З1 — Применить дизайн-систему из сессии 1.1**
Обновить цвета, типографику, радиусы, отступы во всех компонентах Комнаты согласно зафиксированным переменным. Создать CSS-переменные (`:root`) для всей палитры.

**З2 — CornersSection: layout в одну строку**
Реализовать решение из сессии 1.2 — "стена X перекрывает + поле + градус" умещается на 390px.

**З3 — TileForm: цвет шва**
Растянуть color picker до ширины других полей согласно решению из 1.2.

**З4 — WallCard: кнопка "Переопределить плитку"**
Заменить длинную текст-ссылку на кнопку из дизайн-системы согласно решению из 1.2.

**З5 — SummarySection: выравнивание значений**
Реализовать grid-layout из сессии 1.2.

**З6 — WallCard: удаление с подтверждением**
Реализовать решение из сессии 1.2 (swipe / confirm dialog / другое).

**З7 — Нижний таббар с иконками**
Добавить иконки в навбар согласно сессии 1.1. Иконки — SVG инлайн или lucide-react.

**З8 — Валидация размеров**
Не допускать 0 и отрицательные значения для length/height стен и размеров плитки. Показывать inline-ошибку под полем.

**З9 — Прокрутка к маске после добавления**
После `addMask()` — прокрутить к новой маске (`scrollIntoView`).
