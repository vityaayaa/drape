## Промпт

Прочитай файл `docs/sessions/state.md` — там контекст проекта, дизайн-система Dark Glow и **результат сессии 3.1** (split-screen layout, развёртка стен).

Ты дизайнер мобильного PWA **Drape**. Продолжаем работу над вкладкой **«Схема»** — нижняя прокручиваемая часть и диалог экспорта SVG.

**Прочитай для контекста:**
- `planning/EXPORT.md` — требования к легенде, спецификации, экспорту SVG

**Задачи сессии 3.2 — легенда и экспорт:**

1. **Шапка нижней части**: параметры проекта одной строкой над легендой.
   - Что показывать? (размер плитки, ширина шва, цвет шва, итоговые размеры стен)
   - Как скомпоновать на узком экране?

2. **Поле «Запас, %»**: пользователь вводит процент для расчёта плиток с запасом.
   - Где живёт это поле (в шапке? над списком цветов? отдельная строка)?
   - Стиль: инпут или степпер (+/−)?
   - Значение по умолчанию: 10%.

3. **Легенда и спецификация**: список цветов.
   - Как выглядит одна строка: квадратик цвета + номер + hex-код + «N шт.» + «с запасом: M шт.»?
   - Разбивка по стенам под каждым цветом — показывать сразу или раскрывать по тапу?
   - Как сортировать цвета (по убыванию количества? по номеру)?

4. **Диалог настроек экспорта SVG**: появляется по кнопке «Скачать SVG».
   - Это bottom sheet или модальное окно?
   - Две опции масштаба: «На экран» / «Реальный размер 1:1»
   - Переключатель «Рисовать швы» (неактивен при grout_width = 0)
   - Кнопка подтверждения «Скачать»
   - Покажи макет диалога.

5. **Состояние загрузки**: генерация SVG для большой комнаты может занять секунду.
   - Что показывать во время генерации (спиннер на кнопке? overlay?)?
