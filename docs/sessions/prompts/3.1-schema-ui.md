## Промпт

Прочитай файл `docs/sessions/state.md` — там контекст проекта и **вся дизайн-система Dark Glow** (сессии 1.1–1.5).

Ты дизайнер мобильного PWA **Drape**. Сегодня работаем над вкладкой **«Схема»**.

Вкладка «Схема» — это финальный результат для пользователя: развёртка всех стен с цветными плитками и спецификация (сколько плиток какого цвета купить). Сейчас вкладка — заглушка с бейджем «В разработке».

**Прочитай для контекста:**
- `src/components/export/ExportTab.jsx` — текущая заглушка
- `src/store/projectStore.js` — структура данных (стены, `tileColors`, плитка)
- `planning/EXPORT.md` — требования к разделу «Схема»

**Что есть в данных:**
- `walls[]` — список стен, у каждой `width_mm`, `height_mm`, `tile_width`, `tile_height`, `grout_width`, `grout_color`, маски (прямоугольные вырезы)
- `tileColors` — объект `wallId → { 'col_row': '#rrggbb' }` — цвет каждой плитки
- `tile` — глобальные параметры плитки (размер, шов)

**Задачи сессии 3.1 — layout и развёртка:**

1. **Split-screen layout**: экран делится на две части — развёртка стен сверху, легенда снизу. Граница между ними перетаскивается вверх/вниз.
   - Какие пропорции по умолчанию? (50/50 или 60/40?)
   - Как выглядит разделитель (handle)?
   - Есть ли минимальная высота каждой части?

2. **Верхняя часть — развёртка стен**: стены разложены горизонтально в одну строку, каждая стена — прямоугольник с сеткой плиток.
   - Как масштабировать стены? (по высоте экрана, или все одинаковой высоты, или пропорционально реальным размерам?)
   - Зазор между стенами — какой стиль и размер?
   - Название стены — где показывать (над? под? внутри)?
   - Маски-препятствия — как выглядят (пустые белые прямоугольники? затемнённые? штриховка?)?

3. **Плитка в развёртке**: каждая клетка = залитый цветом прямоугольник + номер цвета.
   - Номер цвета — числовой индекс (1, 2, 3...) или буква? Где в клетке?
   - При мелких плитках (много клеток) номер не влезет — что делать? (показывать только при зуме, убирать совсем при мелком размере?)
   - Цвет шва — фон между плитками (цвет шва из настроек) или фиксированный?

4. **Жесты на развёртке**: пользователь должен зумить и двигать развёртку пальцами.
   - Независимый скролл/зум от нижней части?
   - Кнопка «вписать в экран» (reset zoom)?

5. **Пустое состояние развёртки**: что показывать, если пикселизация ещё не запущена (нет `tileColors`)?

6. **Toolbar вкладки «Схема»**: нужна ли верхняя панель? Что на ней?
   - Кнопка «Скачать SVG» — где она живёт (toolbar сверху? кнопка снизу? FAB?)?
