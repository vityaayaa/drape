### 1.1 Формула колонок в schemaRenderer / schemaSVGBuilder

**Проблема:** схема показывает в ~10 раз меньше плиток чем есть на самом деле.

`resolveWallTile` возвращает `tileW`, `groutW` в **0.1 мм** (умножено на 10).  
Но `wallW_mm = parseFloat(wall.length) * 10` — это тоже 0.1 мм.  
Сравни с `roomGeometry.js:109` где всё в мм: `columns = Math.ceil((gridWidthMm + gw) / (tw + gw))`.

Прочитай `src/utils/schemaRenderer.js` и `src/utils/schemaSVGBuilder.js` целиком.  
Найди где именно единицы расходятся и исправь так, чтобы количество колонок совпадало с `roomGeometry.js` для одинаковых входных данных.

**После исправления:** добавь тест в `src/utils/schemaRenderer.test.js` — для стены 300 см, плитка 20 мм, затирка 2 мм. Сравни результат с тем что возвращает `calculateGrid` из `roomGeometry.js` для тех же параметров.

Файлы: `src/utils/schemaRenderer.js`, `src/utils/schemaSVGBuilder.js`

### 1.2 byColor режим укладки сломан (palette = null)

**Файл:** `src/components/layout/LayoutTab.jsx`

Найди строку `rebuildSequence(walls, tile, tileColors, null)`.  
`null` вместо palette → `hexToIndex` возвращает `undefined` для всех плиток → `byColor` сортировка = `byRow`.

**Исправление:**
```js
import { buildPalette } from '../../utils/schemaRenderer.js'

// В теле компонента добавить useMemo:
const palette = useMemo(
  () => buildPalette(walls, tileColors),
  [walls, tileColors]
)

// В useEffect где rebuildSequence — заменить null на palette:
rebuildSequence(walls, tile, tileColors, palette)
```

Убедись что `palette` в deps массиве useEffect вокруг `rebuildSequence`.

### 1.3 tile_overrides игнорируются при пикселизации

**Файлы:** `src/components/pixelizer/PixelizerTab.jsx` (~строки 273–279), `src/components/pixelizer/WallCanvas.jsx` (~строки 129–136)

Сейчас `tileGrid` собирается из `parseFloat(tile.tile_width)` — глобальные размеры без учёта per-wall overrides.

**Исправление:** использовать `resolveWallTile(wall, tile)` из schemaRenderer.js. Помни: `resolveWallTile` возвращает 0.1-мм, нужно делить на 10 для мм:
```js
import { resolveWallTile } from '../../utils/schemaRenderer.js'

const resolved = resolveWallTile(wall, tile)
const tw = resolved.tile_width / 10   // 0.1мм → мм
const th = resolved.tile_height / 10
const gw = resolved.grout_width / 10
```
