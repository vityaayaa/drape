## Промпт

Прочитай файл `docs/sessions/state.md` — там контекст проекта, полный список проблем и **результаты сессии 1.1** (дизайн-система, которую нужно применить).

Ты дизайнер мобильного PWA **Drape**. Сегодня работаем над вкладкой **«Комната»**.

**Прочитай эти файлы для понимания текущей реализации:**
- `src/components/room/RoomTab.jsx`
- `src/components/room/WallCard.jsx`
- `src/components/room/TileForm.jsx`
- `src/components/room/CornersSection.jsx`
- `src/components/room/SummarySection.jsx`
- `src/components/room/MaskCard.jsx`

**Задачи (разбери каждую и предложи решение):**

1. **CornersSection**: поле "стена X перекрывает + градус" не влезает в одну строку на 390px — предложи layout
2. **TileForm**: поле "Цвет шва" узкое (маленький квадрат) — растянуть как остальные поля
3. **WallCard**: "Переопределить плитку для этой стены" — длинная текст-ссылка неудобна на mobile. Предложи нормальную кнопку в стиле дизайн-системы из сессии 1.1
4. **SummarySection**: значения (количество колонок/рядов) не выровнены под своими заголовками — предложи grid-layout
5. **WallCard**: кнопка удаления стены (×) — маленькая, нет подтверждения. Как сделать безопасно и удобно?
6. **MaskCard**: мелкие поля координат — предложи как улучшить readability
7. **Общее**: применить палитру, типографику, радиусы из сессии 1.1 ко всем компонентам Комнаты

Для каждой задачи: покажи **до/после** в виде ASCII-схемы или описания layout. Обсуди со мной перед финализацией.
